# Crown Evaluation Framework

**For DMs only.** No player-facing algorithm. Supports [[The Crown]] philosophy without answering [[open questions]].

---

## Lenses (Observe, Don't Score)

After significant sessions, note which lenses *sparked* for each candidate:

| Lens                   | Question you ask yourself                      |
| :--------------------- | :--------------------------------------------- |
| **Growth**             | Did they change when shown new information?    |
| **Self-awareness**     | Did they name their flaw without prompting?    |
| **Empathy**            | Did they act for someone with less power?      |
| **Responsibility**     | Did they accept cost without blaming the Hall? |
| **Capacity to change** | Did they reverse a harmful pattern?            |

Use [[Candidate Progress Tracker]] — qualitative notes, not numbers fed to Crown.

---

## Challenge Mapping

| Challenge type | Lenses highlighted |
|:--|:--|
| Cooperation | Empathy, Responsibility |
| Public judgment | Self-awareness, Growth |
| Private confession | Self-awareness, Capacity to change |
| Resource allocation | Responsibility, Empathy |
| Truth-telling | Growth, Trust (reputation) |

See [[Choosing Challenges]].

---

## Finale (Session 14)

- Crown may pose **one** question tailored to table themes — examples (choose one, don't publish list to players as canon):
  - *"Who did you become when no one was watching?"*
  - *"Whose suffering did you ignore?"*
  - *"What would you undo?"*
- **Winner:** table decision guided by campaign arcs + player choice. Vault does not name them.

---

## Anti-Patterns

- "Crown points at PC because highest Influence"
- Secret Crown villain arc
- Crown eliminates candidates magically (withdrawal is narrative, not execution)

# Reflection Chambers

**Master:** [[Saint Vaelor's Hall]] · **Index:** [[Hall Locations Compendium]]

## Description

Series of small stone cells with skylights; simpler than suites. Used for voluntary fasting, silence, post-challenge processing.

## Purpose

Reflection after public tests; pairs the [[content/Player/World/Locations/The Quiet Room]] tradition.

## Hall Behaviors

Candles burn evenly only when sitter speaks truth aloud (even if alone).

## NPC Uses

[[Mother Elira Voss]] recommends; [[Dame Lysa Fairward]] uses for judgment meditation.
